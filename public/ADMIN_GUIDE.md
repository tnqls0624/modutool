## ➕ AI 툴 추가하기

### **Step 1: 기본 정보 입력**

#### **툴 ID** (필수)
- 형식: 영문 소문자 + 하이픈
- 예: `chatgpt`, `midjourney`, `stable-diffusion`

#### **툴 이름** (필수)
- 예: `ChatGPT`, `Midjourney`

#### **아이콘** (필수)
- **방법 1**: 파일 업로드 (권장)
  - [파일 선택] 클릭 → 이미지 선택
  - 지원: PNG, JPG, SVG
  - 최대: 2MB
  
- **방법 2**: URL 입력
  - [URL 입력] 클릭
  - 이미지 URL 입력 또는 이모지 (💬)

#### **카테고리** (필수, 다중선택)
- 해당하는 모든 카테고리 체크
- 예: ChatGPT → ✅ 문서 작성, ✅ 개발/코딩

#### **가격 타입** (필수, 다중선택)
- ✅ 무료: 완전 무료
- ✅ 무료 체험: 무료 플랜 + 유료 플랜
- ✅ 유료: 유료만

---

### **Step 2: 활용 사례 추가** (선택) ⭐ **UPDATED!**

#### **타입 선택** (라디오 버튼)
1. **🎬 영상 (YouTube URL)** 선택
   - YouTube URL 입력 필드 표시
   - URL 입력 후 **[➕ 추가]** 클릭
   
2. **🖼️ 이미지 (파일 첨부)** 선택
   - 파일 선택 버튼 표시
   - 이미지 파일 선택 (PNG, JPG, GIF, WebP)
   - 미리보기 확인
   - **[➕ 추가]** 클릭

#### **영상 URL 추가:**
```
1. "영상 (YouTube URL)" 라디오 버튼 선택
2. YouTube URL 입력
   예: https://www.youtube.com/watch?v=xxxxx
3. [추가] 버튼 클릭
4. 목록에 🎬 영상 아이콘과 함께 표시됨
```

#### **이미지 파일 추가:**
```
1. "이미지 (파일 첨부)" 라디오 버튼 선택
2. [파일 선택] 클릭
3. 이미지 선택 (최대 2MB)
4. 미리보기 확인
5. [추가] 버튼 클릭
6. 목록에 🖼️ 이미지 아이콘과 함께 표시됨
```

#### **지원 형식:**
- **영상**: YouTube URL만 지원
- **이미지**: PNG, JPG, GIF, WebP (최대 2MB)

#### **권장:**
- 최대 4개 추가 권장
- 영상과 이미지 혼합 가능
- 2x2 그리드로 표시됨

---

### **Step 3: 추천 타겟 추가** (선택)

1. 추천 대상 입력
2. **[➕ 추가]** 버튼 클릭

**예시:**
```
디자이너 & 아티스트
영상 크리에이터
개발자
```

### **Step 4: 가격 플랜 추가** (선택) ⭐ **UPDATED!**

#### **플랜 정보 입력:**
- **플랜명**: Basic, Pro, Enterprise
- **가격 입력** (최소 1개 필수):
  - 💳 **월 구독 가격**: 월별 반복 결제 금액 (예: $20, ₩10,000)
  - 📅 **연 구독 가격**: 연간 결제 금액 (예: $200, ₩100,000)
  - 🎁 **무료/기타 가격**: 무료, 일회성, 평생 등 (예: 무료, 평생 $299, 문의)
- **플랜 설명**: (선택) 이 플랜의 주요 특징
- **플랜 기능**: 여러 개 추가 가능
  - 예: "GPT-4 모델 사용"
  - 예: "월 100장 이미지 생성"
- **추천 플랜 강조**: ⭐ 체크 시 월 구독 플랜이 파란색 그라데이션 배경으로 표시

#### **플랜 추가:**
1. 플랜명과 가격 정보 입력 (월/연/기타 중 최소 1개)
2. 플랜 설명과 기능 추가 (선택)
3. **[➕ 가격 플랜 추가]** 클릭
4. 입력한 가격에 따라 자동으로 플랜 생성:
   - 월 구독 가격 입력 → 월 구독 플랜 생성
   - 연 구독 가격 입력 → 연 구독 플랜 생성
   - 무료/기타 입력 → 해당 플랜 생성

#### **실전 예시: ChatGPT 가격 플랜**

**예시 1: 월/연 구독 함께 등록**
```
입력:
플랜명: Plus
월 구독 가격: $20
연 구독 가격: $200
플랜 설명: 프리미엄 기능과 우선 접속 제공
플랜 기능:
  - GPT-4 모델 사용
  - 우선 접속권
  - DALL-E 3 이미지 생성
추천 플랜: ✅

→ 자동 생성:
  ✓ Plus (월 구독) - $20 ⭐ (추천 플랜)
  ✓ Plus (연 구독) - $200
```

**예시 2: 무료 플랜만 등록**
```
입력:
플랜명: Free
무료/기타 가격: 무료
플랜 기능:
  - GPT-3.5 모델 사용
  - 제한된 응답 속도

→ 자동 생성:
  ✓ Free (영구 무료) - 무료
```

**예시 3: 월 구독만 등록**
```
입력:
플랜명: Pro
월 구독 가격: $30
플랜 기능:
  - 모든 프리미엄 기능
  - 무제한 사용

→ 자동 생성:
  ✓ Pro (월 구독) - $30
```

💡 **Tip**: 
- 한 번의 입력으로 월/연 구독 플랜을 함께 생성할 수 있습니다!
- 연 구독 가격을 입력할 때 플랜 설명에 할인율을 명시하면 좋습니다
- 추천 플랜은 월 구독 플랜에만 적용됩니다

## 🔧 문제 해결

### ❓ **추가 버튼이 작동하지 않아요**

**해결 방법 1: 브라우저 콘솔 확인**
1. F12 키 누르기
2. Console 탭 확인
3. 오류 메시지 확인

**해결 방법 2: 페이지 새로고침**
```
Ctrl + F5 (강력 새로고침)
```

**해결 방법 3: 입력값 확인**
- 빈 값은 추가되지 않음
- 입력 후 추가 버튼 클릭

---

### ❓ **저장이 안 돼요**

**필수 항목 확인:**
- [ ] 툴 ID 입력
- [ ] 툴 이름 입력
- [ ] 아이콘 등록 (파일 또는 URL)
- [ ] 카테고리 최소 1개 선택
- [ ] 가격 타입 최소 1개 선택
- [ ] 웹사이트 URL 입력
- [ ] 짧은 설명 입력

---

### ❓ **이미지 업로드가 안 돼요**

**파일 크기 확인:**
- 최대 2MB까지 가능
- 권장: 100KB 이하

**파일 형식 확인:**
- PNG, JPG, GIF, SVG, WebP만 가능

**대안:**
- [URL 입력] 사용
- 이미지를 온라인 호스팅 (imgur.com)

---

### ❓ **등록한 툴이 메인에 안 보여요**

**상태 확인:**
- Admin → 툴 목록
- 상태가 "활성"인지 확인
- "임시저장"은 표시 안 됨

**캐시 삭제:**
```
Ctrl + Shift + R
```

## 💡 팁 & 트릭

### ✅ **효율적인 등록 방법**

1. **아이콘 준비**
   - [Flaticon](https://flaticon.com) 에서 다운로드
   - 512x512px PNG 권장

2. **카테고리 선택**
   - 툴의 모든 용도 체크
   - 예: ChatGPT → 문서, 코딩, 번역

3. **가격 플랜 전략** ⭐ **NEW!**
   - 공식 웹사이트에서 최신 가격 확인
   - 월 구독 + 연 구독 함께 등록
   - 연 구독의 할인율 명시
   - 가장 인기있는 플랜을 "추천 플랜"으로 강조
   - 무료 플랜이 있다면 먼저 추가

4. **활용 사례**
   - 실제 결과물 이미지 URL
   - YouTube 리뷰 영상 URL

### 📊 가격 플랜 입력 모범 사례

**❌ 피해야 할 방법:**
```
플랜명: Pro
가격: 20달러/월 또는 200달러/년
```
→ 하나의 플랜에 여러 가격을 섞어서 입력

**✅ 권장 방법:**
```
플랜 1:
플랜명: Pro Monthly
가격: $20
주기: 월 구독

플랜 2:
플랜명: Pro Annual
가격: $200
주기: 연 구독
설명: 연간 구독 시 $40 절약 (20% 할인)
```
→ 각 플랜을 독립적으로 등록
